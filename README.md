# Code Generation Agent

An AI-powered code generation agent built using:

- FastAPI
- Gemini API
- Python
- Tool-based architecture

## Features

- Code generation
- Streaming responses
- File creation tools
- Workspace management

## Tech Stack

- FastAPI
- Gemini
- Python
- LangGraph (planned)
